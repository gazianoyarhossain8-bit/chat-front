#  Real-Time Chat App

A real-time chat application built using React and Socket.IO.

##  Features
- Real-time messaging
- Unique user ID
- Clean UI (WhatsApp style)

##  Tech Stack
- React (Vite)
- Socket.IO
- Node.js (Backend)
